# Emacs / Doom Emacs Keybind Cheat Sheet

## File & Search
- `SPC .` -> open file
- `SPC SPC` -> fzf
- `SPC *` -> ag search

## File Actions
- `SPC f s` -> save

## Navigation
- `SPC w w` -> switch windows
- `g t` -> next tab
- `g T` -> previous tab

## Tools / Panels
- `SPC o t` -> open terminal here
- `SPC o p` -> open sidebar

## Code & Development
- `SPC c d` -> go to definition
- `SPC c c` -> compile
- `SPC c x` -> list errors

## Emacs Launch Options
- `emacs -mm` -> open Emacs maximized
- `emacs -nw` -> open Emacs in terminal
